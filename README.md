## Para ver la web localmente sigue estos pasos:

Clonar el proyecto

```bash
  git clone https://github.com/DevShikzia/bonsai-shop-Sanchez.git
```

Entrar en la carpeta del proyecto

```bash
  cd bonsai-shop
```

Instalar las dependencias

```bash
  npm install
```

Levantar el proyecto

```bash
  npm run start
```
